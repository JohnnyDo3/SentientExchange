## INSPECTION REQUIREMENTS

### Inspections Needed:
1. **Final Inspection** (after installation complete)

### Items Inspector Will Check:
- ✓ Equipment matches permit (model numbers)
- ✓ Proper installation per manufacturer specs
- ✓ Nameplate visible and accessible
- ✓ Electrical disconnect present and labeled
- ✓ Return air properly sized
- ✓ Air filter installed and accessible
- ✓ Thermostat functioning
- ✓ System running and cooling/heating
- ✓ Refrigerant lines insulated
- ✓ Condensate drain properly terminated
- ✓ Permit card posted at job site

### Common Reasons for Inspection Failure:
- Wrong equipment installed (doesn't match permit)
- Missing or unlabeled electrical disconnect
- Condensate drain terminating on neighbor's property
- Refrigerant lines not insulated
- Return air undersized
- System not operational during inspection

---

## NOTES FOR CONTRACTOR

1. **Verify Equipment Match**: Ensure outdoor unit + indoor unit are AHRI-certified matched pair
2. **Existing Circuit**: Confirm existing electrical circuit and breaker are adequate for new equipment
3. **Refrigerant Lines**: Reuse existing lines if in good condition, or replace if corroded/damaged
4. **Condensate Drain**: Verify existing drain is clear and functional
5. **Ductwork**: Inspect existing ductwork for leaks, damage, and proper sizing
6. **Permit Posting**: Print and post permit at job site before starting work
7. **NOC**: If project cost >= $15k, record Notice of Commencement before first inspection
